# 0002. Core Tech Stack

**Status:** Accepted
**Date:** 2026-04-27

## Context

Need to choose tech stack for AI-native platform.

## Decision

- **Framework:** Next.js 15.5 App Router
- **UI library:** React 19.1
- **Language:** TypeScript 5.9.3 (strict)
- **Styling:** Tailwind CSS 4
- **API:** tRPC v11
- **ORM:** Drizzle
- **Database:** Postgres (via Supabase)
- **Auth:** Supabase Auth
- **AI:** Anthropic SDK (exact-pinned)
- **Analytics:** PostHog
- **Hosting:** Vercel
- **Mobile:** Expo (React Native)

## Rationale

Each choice documented in subsequent ADRs (0003-0010).

This ADR establishes the high-level stack alignment.
