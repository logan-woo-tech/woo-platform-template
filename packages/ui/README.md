# @template/ui

Design tokens and UI components.

## Tokens

- `brand` — Brand colors (customize per project)
- `skill` — Skill taxonomy colors
- `semantic` — Success/warning/error/info
- `gray` — Gray scale

## Components

To be added in Sub-phase E:

## Usage

```typescript
import { brand, skill } from '@template/ui';
```
